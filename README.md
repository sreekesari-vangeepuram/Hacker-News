# Hacker-News
This project scrapes the "Hacker News" web-site to retrive the trending news from the site.

<h6>First-steps</h6>:
- In Terminal (MacOS/ Linux):
  ```./hacker_news.py```
- Windows users can use ```python3``` tool in Powershell.
