# Hacker-News
This project scrapes the "Hacker News" web-site to retrive the trending news from the site.

<h5>First-steps</h5>:
In Terminal (MacOS/ Linux):<br>
  ```./hacker_news.py```<br>
Windows users can use ```python3``` tool in Powershell.<br>
